## Installation

To install PPHLX on macOS using Homebrew, run the following commands:

```bash
# Add the custom tap
brew tap pphlx/tap

# Trust the custom tap (required by Homebrew security policies for custom taps)
brew trust pphlx/tap

# Install the compiler
brew install pphlx
```
